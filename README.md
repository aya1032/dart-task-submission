# Task 1 - OOP Account System
This task demonstrates a simple Bank Account system in Dart using Object-Oriented Programming concepts.  
Features:
- Base class `Account` with `ownerName`, `balance`, `deposit()`, `withdraw()`.
- Subclass `SavingsAccount` with `interestRate` and `addInterest()` method.
- Subclass `CheckingAccount` with `withdrawalFee` applied on withdrawals.
- Demonstrates inheritance, method overriding, and polymorphism.
